Class for define the user address for a better located ad experience

## Methods

### `getCity()`

**Return:** String.

### `getState()`

Return the user State

**Return:** String.

### `getCountry()`

Return the user Country

**Return:** String.

### `getLatitude()`

**Return:** double.

### `getLongitude()`

**Return:** double.
